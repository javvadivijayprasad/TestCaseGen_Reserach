## How to cite

If you use the **RAITG** dataset, pipeline, or results, please cite the archived deposit on Zenodo.

### Concept DOI (cite this for general references)

```bibtex
@dataset{javvadi_raitg_2026,
  author    = {Javvadi, Vijay Prasad},
  title     = {{RAITG}: Requirement-Aware Intelligent Test Generator --- Dataset, Pipeline, and Reproducibility Bundle},
  year      = 2026,
  publisher = {Zenodo},
  doi       = {10.5281/zenodo.20285103},
  url       = {https://doi.org/10.5281/zenodo.20285103}
}
```
